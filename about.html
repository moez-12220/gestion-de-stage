À Propos d'InternShip Manager

Notre Mission

InternShip Manager est une plateforme universitaire développée dans le cadre d'un projet académique de développement web. Notre objectif est de simplifier et moderniser la gestion des stages pour les étudiants, les entreprises et les enseignants.

Trop souvent, la gestion des stages est fragmentée — formulaires papier, emails dispersés, suivi manuel. Notre plateforme centralise tous ces processus dans une interface unique, intuitive et accessible.

Pour toute question, contactez-nous à : contact@internship-manager.tn

Retrouvez la documentation et le code source sur notre dépôt GitHub.
