# 🎓 Gestion de Stage — InternShip Manager
 
Application web de gestion des stages — suivi des stagiaires, entreprises et conventions de stage.
 
---
 
## 📋 Description
 
**InternShip Manager** est une application web statique développée dans le cadre du projet ENSI 2025/2026. Elle permet de gérer et suivre les stages : consultation des offres, espace étudiant, espace entreprise, et plus encore.
 
---
 
## 🗂️ Structure du projet
 
```
site/
├── index.html               # Page d'accueil
├── about.html               # À propos
├── contact.html             # Page contact
├── offres.html              # Offres de stage
├── espace-etudiant.html     # Espace étudiant
├── espace-entreprise.html   # Espace entreprise
├── funpage.html             # Page fun
├── questionnaire.html       # Questionnaire
├── styles.css               # Feuille de style principale
├── logo.jpg                 # Logo du site
└── js/
    ├── contact.js           # Script page contact
    ├── espace-etudiant.js   # Script espace étudiant
    ├── funpage.js           # Script page fun
    └── questionnaire.js     # Script questionnaire
```
 
---
 
## 🚀 Technologies utilisées
 
- **HTML5** — Structure des pages
- **CSS3** — Mise en forme et design
- **JavaScript** — Interactions dynamiques

1. Clone le dépôt :
   ```bash
   git clone https://github.com/moez-12220/gestion-de-stage.git
   ```

   ```bash
   cd gestion-de-stage
   ```
 
3. Ouvre `index.html` dans ton navigateur.
